# 校园志愿者管理系统 java936

## 项目概述

校园志愿者管理系统是一款基于前后端分离架构的信息管理系统，旨在高效地管理校园志愿者活动及相关信息。

## 技术栈

- **前端：**
  - Vue
  - Element-UI
- **后端：**
  - Spring Boot
  - MyBatis-Plus
- **开发语言：**
  - Java
- **数据库：**
  - MySQL
- **运行环境：**
  - IDEA 或 Eclipse
  - VSCode

## 功能模块

### 系统角色

- **管理员：**
  - 登录
  - 首页

  - 志愿者管理
  - 活动类型管理
  - 活动信息管理
  - 活动报名管理
  - 活动通知管理
  - 活动心得管理
  - 交流反馈
  - 系统管理

- **用户：**
  - 登录
  - 注册
  - 首页
  - 活动信息
  - 活动心得
  - 公告信息
  - 交流反馈
  - 个人中心
  - 我的收藏
  - 活动报名管理
  - 活动通知管理
  - 活动心得管理
